This branch's changed contents are in the game directory, to play this mod you will need to pull this branch and make a solution 
for it in visual studio (or a similar app). Simply build it and have it sent to a seprate folder in the quake2 files.
After load quake2 and input the command +set game "foldername".

Further you will also need a different config file for the new folder provided by this branch in game (the dll is also present here)

This mod turns quake into a turn based rpg, it's guis operate on a button basis as all the options are labeled at the end of each option
This follows suit for every screen
Simply fire the blaster at an enemy to initate combat, and then select an attack at the cost of bullets.
Please note that you may need to pull up the main screen occasionaly (simply press p).
